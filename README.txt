Repositorio para fins educacionais

Algoritmo desenvolvido pelo Felipe Martins dos Santos
https://github.com/felipemartinsss
https://github.com/felipemartinsss/RepositorioIAsc/tree/master/Python/buscas

O algoritmo realiza buscas a partir de uma matriz, considerando o problema do duende perdido, questao da Olimpíada Brasileira de Informática de 2005 (OBI-2005).
Site da publicação: https://iascblog.wordpress.com/2016/05/10/buscas-em-largura-e-a-star-usando-python/

Meu intuito com esse repo é estudar os tipos de buscas em IA, e aprender suas implementações em python.
Estudo realizado durante o quinto periodo do curso de ciencias da computação na UNIPESU de afogados, Recife-PE no ano de 2022.
